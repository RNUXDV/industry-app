# Industry Design Language

# Foundations

The Foundations define the visual language that every component and experience builds upon.

---

# Color Philosophy

Color communicates meaning before decoration.

Every color should have a purpose.

Examples include:

- Success
- Warning
- Information
- Error
- Neutral
- Selection

---

# Typography

Typography should prioritize readability over expression.

Goals:

- Clear hierarchy
- Comfortable scanning
- Low cognitive load
- Consistent spacing

---

# Spacing

Whitespace is functional.

Spacing should separate ideas before borders are needed.

Users should never feel visually crowded.

---

# Motion

Animation should communicate change.

Never animate simply for decoration.

Motion should:

- Confirm actions
- Guide attention
- Reinforce hierarchy

---

# Elevation

Elevation communicates importance.

Avoid unnecessary shadows.

Use layers sparingly.

---

# Accessibility

Every foundation should support:

- Keyboard navigation
- Screen readers
- Color contrast
- Reduced motion
- Scalable text

Accessibility is part of the design, not an afterthought.

---

# Performance

Interfaces should feel lightweight.

Fast interactions build trust.

Every animation, component, and dependency should justify its cost.
