# Industry Design Language (IDL)

The Industry Design Language defines the principles, components, patterns, and experiences that create a consistent product throughout Industry.

---

## Mission

Create calm, intuitive workplace software that reduces uncertainty and supports service industry professionals.

---

## Core Principle

Every interaction should reduce uncertainty.

---

## Pillars

### Foundations
- Color
- Typography
- Spacing
- Motion
- Accessibility

### Components
- IDL-001 — Status Pill
- IDL-002 — Avatar
- IDL-003 — Presence Card

### Patterns
- Selection
- Approval Workflow
- Empty States
- Progressive Disclosure

### Experiences
- Schedule
- Catch
- Crew
- Jobs
- People
